Feature #60: "Still fetching..." after 5 seconds

Test Overview

Feature: If loading exceeds 5 seconds, show "Still fetching..." message to reassure users.

Implementation:

  • Added showStillFetching state to useWeather hook
  • Set 5-second timeout in fetchWeather function
  • Updated WeatherSkeleton to display friendly message
  • Message includes suggestion to check internet connection

Live Test

To test this feature, we need to simulate a slow API response:

  1. Open browser DevTools (F12)
  2. Go to Network tab
  3. Select "Slow 3G" throttling
  4. Reload the app below
  5. Wait 5+ seconds
  6. Verify blue "Still fetching..." message appears
